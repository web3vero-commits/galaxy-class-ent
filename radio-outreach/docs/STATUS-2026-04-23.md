## Rebuild instructions (if seed needs to be regenerated)

1. FCC LMS bulk dump: `https://enterpriseefiling.fcc.gov/dataentry/api/download/dbfile/Current_LMS_Dump.zip` (1.68 GB) OR individual `facility.zip` + `application_facility.zip` + `app_party.zip` at dated subpath `/dbfile/MM-DD-YYYY/`
2. Filter `facility.dat` to `service_code IN ('AM','FM') AND facility_status = 'LICEN'`
3. Join `facility.facility_id = application_facility.afac_facility_id → app_party.apar_aapp_application_id WHERE apar_applicant_ind = 'A'`
4. Nielsen ranks: scrape `https://ratings.radio-online.com/content/arb_menu_rank`
5. CBSA crosswalk: Census Gazetteer 2024 `places` + `counties` + OMB CBSA `list1_2023.xlsx`
6. Owner canonicalization + category tagging + calling window logic — see session history
